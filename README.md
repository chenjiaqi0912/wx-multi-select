# wx-multi-select
微信小程序多选组件


主要实现通过Js+wxml+wxss+vant实现点击一组复选框，动态添加1个input输入框，输入框中自动填充所选复选框的内容,可以在wxml循环中限制个数
同理，使用js、html、css同样可以实现。
